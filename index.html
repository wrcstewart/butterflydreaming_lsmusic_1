SIMPLE EXAMPLE OF TEXT TO MEDIA — ButterflyDreaming Platform

This page demonstrates how a text node from the ButterflyDreaming graph can drive a media module. In the live platform the L-system notation and %%bd_ directives shown below would be found in a graph node, discovered and edited collaboratively by two anonymous users during a dyadic encounter.

The %%bd_ directives are a shared platform language — each directive is available to all media modules, which are free to interpret them in their own way or ignore them silently. A directive that controls reverb in a music module might influence colour or motion in a visual module.

In ButterflyDreaming the creative process is always collaborative. New nodes are produced by merge-editing of ancestor nodes, so the set of directives a user encounters grows gradually through inheritance rather than being invented from scratch.
